## 🚀 Quick Start

### Frontend

1. Navigate to `frontend/` directory
2. Run `npm install`
3. Run `npm run dev`
4. Visit `http://localhost:5173`

### Backend

1. Navigate to `backend/` directory
2. Run `npm install`
3. Set up MongoDB connection
4. Run `npm run dev`
5. API available at `http://localhost:3000`
